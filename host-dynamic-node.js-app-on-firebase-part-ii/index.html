Host dynamic Node.js app on Firebase: Part-II

Firebase Cloud-functions
Last updated: Apr 6, 2020

In this article we will connect our app with Firestore database.

TL,DR;

In the previous article we built a simple express app and deployed it to Firebase. To continue with this article you should read about previous one.

Steps to connect and get data from Firebase real-time database.

1)Include firebase-admin SDK to the project

const admin = require('firebase-admin');
admin.initializeApp(); //creates a firebase app
const topics = admin.database().ref('/topics'); //reference to the collection or table

2)Create a firebase project and import a collection

Go to firebase console and create a project.

In the database section select real-time database and create a collection.

I am creating a topics collection in this tutorial.

3)Fetch data from collection

Use below function to fetch data from topics collection. It returns data as a promise.

function getTopics(){
    return topics.once('value').then(snap=>snap.val());
}

Change the ‘/’ route in index.js with this.

app.get('/',(req,res)=>{
    let topics = [];
    getTopics().then(data => {
      for (let key in data) {
        topics.push({
            id: key,
            name : data[key].name
          });
        }
      res.render('index',{data:topics});
    });
})

Render the data in template using below code. Just replace it with previous one and get ready to see the magic.
